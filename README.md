# Jogo de Adivinhação em Java

Projeto simples que fiz para treinar lógica de programação usando Java.

## Sobre

O programa escolhe um número e o usuário tenta adivinhar.  
A cada tentativa, ele informa se o número é maior ou menor até acertar.

## O que pratiquei

- Entrada de dados com Scanner  
- Estruturas condicionais (if / else)  
- Laço de repetição (while)  

## Como executar

Compile:
javac Main.java

Execute:
java Main

## Objetivo

Estou estudando programação e criando projetos simples para evoluir e montar meu portfólio.
